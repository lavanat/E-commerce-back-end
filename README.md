# E-commerce-back-end
A back-end database and application for an E-commerce site using mySQL, dotenv, and sequalize

## Description

Users can pull a list of products, tags, and categories for an e-commerce site using express.js and sequelize.

## Usage

You can use this application to demonstrate and start a backend application to an e-commerce site.
